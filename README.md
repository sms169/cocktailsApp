CocktailsApp for Android 

This was vibe-coded using Google Antigravity and Android Studio. 

Functionality 
  1. Ability to search top cocktails by ingredients 
  2. Switch between the CocktailsDB API and Gemini (for llm based suggestions)
  3. In the Cocktails details page, the app shows an image, recipe, some history, rough calories and a link to Youtube for how to make the cocktail 
  4. Ability to favorite recipes 
  5. Ability to search cocktails by name. 

Screenshots in the screenshots/ folder 


To run this app : 
  1. Add a Gemini API key to the secret.properties file in the top level folder.
